# Technology Stack

## Core Frameworks

- **Frontend Framework:** Nuxt 4 (Vue 3)
- **Language:** TypeScript
- **Runtime/Package Manager:** Bun

## Styling & UI

- **CSS Framework:** Tailwind CSS (via `@nuxtjs/tailwindcss`)
- **Map Library:** Mapbox GL JS (via `nuxt-mapbox`)

## Tooling & Configuration

- **Build Tool:** Bun / Nuxt CLI
- **Linting/Formatting:** Standard Nuxt/Vue tooling (inferred)
